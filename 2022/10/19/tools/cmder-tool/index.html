# 常用快捷键

快捷键说明
Tab自动路径补全
Ctrl+T建立新页签
Ctrl+W关闭页签
Ctrl+Tab切换页签
Alt+F4关闭所有页签
Alt+Shift+1开启 cmd.exe
Alt+Shift+2开启 powershell.exe
Alt+Shift+3开启 powershell.exe (系统管理员权限)
Ctrl+n快速切换到第 n 个页签 (n 值无上限)
Alt + enter切换到全屏状态
Ctr+r历史命令搜索
Win+Alt+P开启工具选项视窗
